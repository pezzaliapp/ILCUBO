# ILCUBO

Evoluzione di **KubeApp** (Il Cubo di Rubik PWA): personalizzazione totale delle facce con colori, numeri, lettere, testi, emoji e immagini.

- **Stato**: in sviluppo (v0.1.0)
- **Autore**: Alessandro Pezzali — [pezzaliAPP.com](https://www.pezzaliapp.com)

Storia, motivazioni e versione storica: vedi [`README.legacy.md`](./README.legacy.md).

## Sviluppo
```bash
# server locale per testare la PWA
python3 -m http.server 8000
# poi apri http://localhost:8000
```
